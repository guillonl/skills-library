Respecter la charte graphique Énergir dans toutes les interfaces produites. La palette principale repose sur le orange Énergir (#F47920) comme couleur d'accent, le bleu foncé (#1A3A5C) pour les éléments de confiance, et des gris neutres pour le contenu secondaire.

Utiliser la typographie corporative (famille sans-serif) avec une hiérarchie claire : titres en bold, sous-titres en medium, corps de texte en regular. Les tailles doivent suivre l'échelle définie dans le design system (14, 16, 18, 24, 32, 48px).

Le ton de voix Énergir est chaleureux, accessible et professionnel. Éviter le jargon technique dans les interfaces client. Privilégier les formulations positives et orientées vers l'action.

Tous les composants doivent intégrer le border-radius standard (8px), les ombres légères pour la profondeur, et respecter les espacements du grid 8px. Valider la conformité avec l'équipe marque avant livraison.
