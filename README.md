# Skills Library

Collection de skills Claude Code organisés par projet client et métier.

## Structure

- `projects/` — Skills spécifiques par client
- `metiers/` — Skills par discipline (Design UI, Graphisme, UX Research, Motion)

## Ajouter un skill

1. Créez un fichier `.md` dans le dossier approprié
2. Ajoutez le frontmatter (name, description, tags, author, date, color)
3. Écrivez le contenu du skill
4. Push sur `main`

Le Skills Hub se met à jour automatiquement.
