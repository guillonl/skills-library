Audit every component, spacing value, color, and typographic choice against the established design system. Identify where the implementation has drifted from defined tokens -- hardcoded values, one-off spacing, custom colors that should be system colors, and components that deviate from documented patterns. Replace all deviations with their correct design system equivalents. The result should be an interface that is fully traceable back to the system's source of truth.
