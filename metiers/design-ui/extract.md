Scan the codebase for repeated UI patterns, duplicated styles, and ad-hoc implementations that should be shared components. Identify candidates for extraction -- recurring card layouts, button variants, form patterns, spacing scales, and color values. Refactor them into well-named, configurable components with clear APIs and consistent token usage. Each extraction should reduce duplication, improve maintainability, and enrich the design system for future use.
