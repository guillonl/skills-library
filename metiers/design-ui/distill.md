name: Distill
description: Strip designs to their essence by removing unnecessary complexity
tags: [simplification, minimal, clean]
author: Impeccable
date: 2026-04-06

---

Systematically remove every element that does not directly serve the user's primary task. Challenge each border, divider, icon, label, and decorative element -- if removing it does not reduce clarity, it should go. Consolidate redundant controls, merge overlapping sections, and reduce cognitive load at every step. Great design is not about adding until it looks good; it is about removing until only what matters remains.
