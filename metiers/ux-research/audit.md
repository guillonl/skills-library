Execute a comprehensive technical audit covering accessibility (WCAG compliance, ARIA usage, keyboard navigation, color contrast), performance (render blocking, layout shifts, image optimization), responsive behavior (breakpoint coverage, touch targets, viewport handling), and theming (dark mode support, token coverage). Score each category and assign severity levels from P0 (critical) to P3 (minor). Produce a prioritized action plan with specific code locations and recommended fixes for every issue found.
