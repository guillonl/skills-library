Run this once per project to establish persistent design context. Gather information about the project's tech stack, design system, component library, color tokens, typography scale, spacing conventions, and brand guidelines. Identify the project's .claude or AI configuration file and write the collected design context there so all future Impeccable skills have the grounding they need. This ensures every design skill operates with awareness of your specific project constraints, patterns, and standards rather than generic defaults.
