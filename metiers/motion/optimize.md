Profile the interface for performance bottlenecks across loading, rendering, and interaction responsiveness. Identify and fix layout thrashing, excessive re-renders, unoptimized images, render-blocking resources, and oversized JavaScript bundles. Ensure animations run on the compositor thread at 60fps, lazy-load off-screen content, and implement proper caching strategies. Measure Core Web Vitals (LCP, FID, CLS) before and after changes to quantify improvements. Performance is a feature -- every millisecond matters.
