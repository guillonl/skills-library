Identify interaction moments that benefit from motion -- state changes, page transitions, content reveals, hover feedback, and loading sequences. Add animations that communicate meaning: easing that feels natural, durations that respect user attention, and choreography that guides focus. Use CSS transitions, keyframe animations, or spring physics where appropriate. Every animation must serve a purpose -- orienting the user, confirming an action, or creating continuity between states. Remove any motion that is purely decorative without functional value.
